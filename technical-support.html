Common Issues & Solutions

Issue Possible Causes Solutions
Device Not Connecting to Wi-Fi/GSM
  • Weak Wi-Fi or GSM signal
  • Incorrect network credentials
  • SIM card not activated or out of balance
  • Check Wi-Fi/GSM signal strength
  • Verify network credentials
  • Restart the device
No Data Appearing on Dashboard
  • Sensor connection issue
  • No internet connectivity
  • Server issue
  • Check sensor connections
  • Test internet connection
  • Restart device and refresh dashboard

Reset & Recalibrate Sensors

Sensor Recalibration Steps
PMS7003 (Particulate Matter Sensor)
  • Power off the device
  • Place in clean air environment for 10 minutes
  • Restart and allow stabilization
MiCS-6814 (Gas Sensor)
  • Place in a ventilated area for 5-10 minutes
  • Power cycle the device
  • Avoid exposure to high humidity and extreme temperatures
SHT41 (Temperature & Humidity Sensor)
  • If inaccurate, restart the device and ensure proper placement
  • If issues persist, replace the sensor
